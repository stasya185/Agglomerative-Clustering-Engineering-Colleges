# Иерархическая кластеризация инженерных колледжей

## Описание задачи
В данном проекте проводится кластеризация 26 инженерных колледжей на основе двух признаков: качество преподавания (Teaching) и наличие стажировок (Internship). 
Задача — объединить похожие учебные заведения в группы с помощью иерархической кластеризации.

## Используемые технологии
- Python
- Pandas 
- Scikit-learn 
- Matplotlib 

## Методы и результаты
В проекте реализованы два подхода к иерархической кластеризации с использованием метода Уорда (Ward's linkage) и евклидова расстояния:
1. **Анализ через SciPy:** 
   - Построена и визуализирована дендрограмма для выявления структуры данных.
   - Произведена разбивка на кластеры с помощью функции `fcluster` (порог расстояния 10).
2. **Анализ через Scikit-learn:**
   - Использована модель `AgglomerativeClustering` для разбиения на **3 кластера**.
   - Построен точечный график, наглядно показывающий распределение колледжей по кластерам.

**Итог:** Оба метода успешно выделили группы колледжей с различными показателями в двухмерном пространстве (преподавание и стажировки), что позволяет увидеть, какие вузы имеют схожий профиль.
